On this page you’ll find a quick reference to common HTML terms and a table of frequently used named character entities.

HTML Terms

Attribute
An HTML element property that provides additional information or functionality for an element.
Block-Level Element
HTML tags that create blocks of content within the document.
Child Element
Any HTML element that is contained inside another HTML element.
Closing Tag
Appears after an element’s content and indicates the end of an element.
Comment
Reference information added to the code.
CMS
Content Management System. Software used to create and maintain content.
CSS
Cascading Style Sheets used to control presentation and layout.
Doctype
Document Type Declaration. Identifies the HTML version and triggers standards mode.
DOM
Document Object Model. A structured representation of a document.
Elements
HTML elements consist of a tag and the content itself.
FTP
File Transfer Protocol. Used for transferring files between servers.
HTTP
Hypertext Transfer Protocol. Controls formatting and transmission of HTML.
Inline-Level Elements
Elements that appear within the normal flow of text.
JavaScript
Scripting language used to control page behavior.
Metadata
Information about the document, usually stored in meta tags.
MIME Type
Identifies the type of file being transferred.
Parent Element
An HTML element that contains another element.
Replaced Element
Elements replaced by external resources like images or video.
Tag
Markup characters that identify content.
URL
Uniform Resource Locator. The address of a resource.
W3C
World Wide Web Consortium. Authors the HTML specification.
